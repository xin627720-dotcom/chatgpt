# 高考英语词汇学习 App（PWA）安装说明

部署后（HTTPS 域名），可以在安卓手机和 iPad Air 2 上“添加到主屏幕”，像 App 一样打开。

## 一、安卓手机安装到桌面（Chrome）

1. 用 Chrome 打开你部署好的网址（例如 `https://xxx.com`）。
2. 正常浏览几秒，确认页面能加载（首页/学单词等）。
3. 点击浏览器右上角菜单（⋮）。
4. 点击 **“添加到主屏幕”** 或 **“安装应用”**（不同机型文案略有不同）。
5. 确认安装后，桌面会出现图标。
6. 以后从桌面图标打开，就是独立窗口，接近原生 App 体验。

## 二、iPad Air 2 安装到桌面（Safari）

1. 用 Safari 打开你部署好的网址（必须是 HTTPS）。
2. 点击下方分享按钮（方框上箭头）。
3. 选择 **“添加到主屏幕”**。
4. 可修改名称（建议“词汇底座”），再点“添加”。
5. 回到主屏幕后会出现图标。
6. 从图标打开时会以全屏 Web App 方式运行。

## 三、注意事项

- 第一版不需要登录。
- 第一版不做多设备同步。
- 学习进度保存在每台设备自己的 `localStorage`：
  - 手机上的学习记录只在手机本机。
  - iPad 上的学习记录只在 iPad 本机。
- 如果你清理浏览器网站数据/缓存，学习记录可能会被清除。

## 四、安装失败排查

1. 确认使用 `https://`（不是 `http://`）。
2. 确认使用系统浏览器：安卓建议 Chrome，iPad 必须 Safari。
3. 打开网页后停留几秒再尝试“添加到主屏幕”。
4. 若刚发布新版本，先刷新页面后再安装。
