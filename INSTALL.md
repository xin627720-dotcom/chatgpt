# 高考英语词汇学习 App 安装到桌面说明

这个项目是个人自用 PWA。部署成 HTTPS 网址后，可以在安卓手机和 iPad Air 2 上添加到主屏幕，像 App 一样打开。

## 安卓手机

1. 用 Chrome 打开部署后的网址。
2. 等首页正常显示。
3. 点右上角三个点。
4. 点“添加到主屏幕”或“安装应用”。
5. 桌面出现图标后，以后直接点图标打开。

## iPad Air 2

1. 用 Safari 打开部署后的网址。
2. 点分享按钮。
3. 点“添加到主屏幕”。
4. 名称可以写“词汇学习”。
5. 点“添加”。

## 注意

- 第一版不需要登录。
- 第一版不同设备之间不自动同步学习记录。
- 学习记录保存在每台设备自己的浏览器本地存储里。
- 如果清理浏览器网站数据，学习记录可能会消失。
